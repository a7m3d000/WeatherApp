# Weather App

By this **Web App** You can get the weather info by just type your Region or city.

# Discreption
This is a weather App that show you Your Region Informations:
  - Weather State
  - Exact Tempreture
  - Min tempreture
  - Max Tempreture
  - Wind Speed
